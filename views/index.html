URL Shortener

This service takes a URL from you and gives a short URL that can be used to open the original one.

Example creation of short URL:

Input :

https://url-shortener-service.glitch.me/new/https://www.google.com

Output :

{"original_url":"https://www.google.com","short_url":"https://url-shortener-service.glitch.me/4"}

Example Usage of short URL:

This URL:

https://url-shortener-service.glitch.me/4

Will Open :

https://www.google.com
